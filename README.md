🌟 Features
✨ Core Features

    Real-time Conversion - Instant results as you type
    📂 12 Categories - Length, Weight, Temperature, Volume, Area, Speed, Time, Data, Energy, Pressure, Angle, Frequency
    🔍 Unit Search - Quickly find any unit
    🔄 Swap Button - Swap source and target units instantly
    Copy Result - One-click copy to clipboard
    📜 Conversion History - Save last 20 conversions
    🧮 Formula Display - Show conversion formula
    Quick Conversions - See multiple conversions at once
    💾 Local Storage - History persists across sessions

Design Features

    🌙 Dark Blue Theme - Eye-friendly dark mode
    Glassmorphism UI - Modern glass-like effects
    ✨ Smooth Animations - Beautiful transitions
    📱 Fully Responsive - Works on all devices
    Toast Notifications - Elegant feedback
    🌈 Gradient Accents - Stunning color gradients

💡 How to Use

    Select Category: Click on any category card (Length, Weight, etc.)
    Enter Value: Type the value you want to convert
    Choose Units: Select source and target units from dropdowns
    View Result: See the converted value instantly
    Swap Units: Click the swap button to reverse conversion
    Copy Result: Click the copy button to copy the result
    View History: Check your last 20 conversions at the bottom
